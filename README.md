# Simple Calculator Program
This is a simple calculator program written in C language that takes two inputs from the user and performs arithmetic operations based on the user's instructions.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Requirements
- C compiler (e.g., GCC)
- Git
